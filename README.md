# AutoCommit
AutoCommit je batch soubor, ktery automaticky commituje a pushuje vsechno ve slozce, ve ktere je na main. Usefull kdyz neumis git.

## Navod na pouziti
Pretahni si nebo zkopiruj AutoCommit.bat do slozky, kde mas projekt. Potom dej do .gitignore:

```.gitignore
AutoCommit.bat
```
Nebo dej AutoCommit.gitignore do slozky projektu taky a odeber komentar aby zacal fungovat.
